# Design System — Database Lab
> "Technical Elegance with Educational Warmth"

## Aesthetic Direction

A production-grade educational interface that balances **precision** (the mindset of a database engineer) with **warmth** (the environment of learning). Every design decision is intentional: confident typography, purposeful spacing, and interactions that reward attention without demanding it.

---

## Color System (OKLCH)

All colors are defined using OKLCH for perceptual uniformity — adjusting lightness never shifts perceived saturation.

| Token | Value | Use |
|---|---|---|
| `--color-primary-500` | `oklch(52% 0.165 220)` | Interactive elements, links |
| `--color-primary-600` | `oklch(44% 0.155 220)` | Buttons, table headers |
| `--color-accent-400`  | `oklch(80% 0.165 72)` | Progress, highlights |
| `--color-success-500` | `oklch(58% 0.150 145)` | Correct answers, pass states |
| `--color-error-500`   | `oklch(58% 0.200 25)` | Incorrect answers, errors |

### What we removed
- ❌ Purple-to-violet gradient (`#667eea → #764ba2`) — generic AI aesthetic
- ❌ Pure grays — all neutrals are tinted toward primary hue (220°)
- ❌ Yellow progress bar — replaced with primary→accent gradient

## Typography

| Role | Font | Weight |
|---|---|---|
| Body text | Inter | 400, 500 |
| Headings | Inter | 700, 800 |
| Code / monospace | JetBrains Mono | 400, 500, 600 |

Both loaded from Google Fonts with `display=swap` for performance.

### Rules
- Headlines: `letter-spacing: -0.025em`, `line-height: 1.25`
- Body: `line-height: 1.75`, `color: --text-secondary`
- Code labels/badges: uppercase, `letter-spacing: 0.05em`
- No widows: `max-width: 60ch` on paragraph containers

## Spacing Scale

Based on 8px grid (`--space-N`). Values: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80px.

## Component Patterns

### Cards (`.content-section`)
- White background, `border-radius: 12px`
- `border-left: 4px solid --color-primary-400`
- Hover lifts with `box-shadow: --shadow-md`

### Buttons (`.btn`)
- `.btn--primary`: filled, primary color, lifts `translateY(-2px)` on hover
- `.btn--outline`: transparent with border, subtle background on hover
- All states: default → hover → active → disabled → loading
- Disabled: `opacity: 0.45`, `pointer-events: none`

### Sidebar
- Dark panel (`--surface-sidebar = oklch(18% 0.025 220)`)
- Active item: left indicator line + tinted background
- Collapsed to 60px icon-only mode

### Header
- Fixed, dark, scroll-aware (deeper shadow when scrolled)
- Progress bar at bottom tracks topic position

### Code Blocks (`.code-block`)
- Dark background (`--surface-code`)
- `border-left: 3px solid --color-accent-400`
- JetBrains Mono, `font-size: 0.875rem`

## Motion & Transitions

| Duration | Use |
|---|---|
| 150ms ease-out | Instant feedback (toggles, checks) |
| 200ms ease-out | Standard (backgrounds, colors) |
| 250ms cubic-bezier(0.34,1.56,0.64,1) | Spring (transforms, scales) |
| 300ms ease-out | Slower reveals |

### Scroll reveals
Elements with `.reveal` class animate `opacity: 0 → 1`, `translateY(16px → 0)` when entering viewport, observed via `IntersectionObserver` in `ContentArea`.

### Confetti
Triggered on quiz scores ≥ 70%. CSS-only animation, 24 pieces, auto-clears after 3s. Fully skippable (`prefers-reduced-motion` respected via global rule).

---

## Accessibility

- **WCAG AA**: All text/background combinations checked at ≥ 4.5:1 contrast
- **Focus states**: `outline: 2px solid --color-primary-500` with `outline-offset: 2px`
- **Keyboard navigation**: All interactive elements are `button` or `a` elements, no `div` click handlers
- **Screen readers**: `aria-label` on icon-only buttons, `aria-current="page"` on active nav item, `role="status"` on loading states
- **Reduced motion**: Global `@media (prefers-reduced-motion: reduce)` collapses all durations to 0.01ms
